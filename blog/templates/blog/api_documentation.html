Posts API


  • Getting Post Lists
    http://localhost:8000/api/post/list

  • Creating New Post
    http://localhost:8000/api/post/new

  • Getting Specific Post Detail
    http://localhost:8000/api/post/post_id

  • Updating A Post [ title and content field must be provided ]
    http://localhost:8000/api/post/post_id/update

  • Deleting A Post
    http://localhost:8000/api/post/post_id/update



Accounts API


  • Register [ username, email, password, password2 must be provided ]
    http://localhost:8000/api/account/register

  • Log In [ username,password must be provided ]
    http://localhost:8000/api/account/login

  • Account Properties
    http://localhost:8000/api/account/properties

  • Update Account Properties [ username , email must be provided ]
    http://localhost:8000/api/account/properties/update
